# 步驟
1. 處理對照用的 A ~ Z、0 ~ 9 圖片
2. 爬取證交所買賣日報表頁面的驗證文字(圖片)
3. 處理並切割驗證文字(圖片)後個別與對照用的字母/數字圖片計算出最小的MSE，以便找出該字母是哪個字
4. 處理資料格式，並輸出成果**買賣日報表.csv**
